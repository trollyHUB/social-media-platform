# 🔒 Security Policy

## Поддерживаемые версии

| Версия | Поддержка |
|--------|-----------|
| 1.0.x  | ✅ Активная |

## Сообщить об уязвимости

Если вы обнаружили уязвимость безопасности, **НЕ создавайте публичный Issue**.

Вместо этого:
1. Отправьте email на: trollyhub@gmail.com
2. Или создайте приватное сообщение через GitHub

### Что включить в отчёт:
- Описание уязвимости
- Шаги для воспроизведения
- Возможное влияние
- Предлагаемое решение (если есть)

### Время ответа:
- Подтверждение получения: 48 часов
- Оценка и план: 7 дней
- Исправление: зависит от серьёзности

## Практики безопасности

- Пароли хешируются через BCrypt
- Аутентификация через JWT
- CORS настроен
- SQL injection защита через JPA/Hibernate
- XSS защита через React
- Секреты НЕ хранятся в репозитории
